# NBA Scoring Prediction Analysis

This project predicts NBA player scoring performance, measured by points per game, using player statistics from the 2024–2025 NBA season.

## Research Question
Can NBA player scoring performance be accurately predicted using efficiency metrics, opportunity measures, and advanced statistics?

## Methods Used
- Linear Regression
- Random Forest Regression
- Repeated train/test splits
- RMSE and MAE evaluation

## Tools
- R
- Tidyverse
- randomForest
- Metrics

## Results
Both models predicted scoring performance with strong accuracy. The linear regression model achieved an average RMSE of about 2.42 and MAE of about 1.80. The random forest model performed slightly better with an RMSE of about 2.25 and MAE of about 1.58.
